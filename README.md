## 🚀 Usage

1. Enter **Title, Price, Taxes, Ads, Discount, Count, and Category**.  
2. Click **Create** to add the product (popup alerts if required fields are empty).  
3. Use the **Update** button to edit a product.  
4. Use the **Delete** button to remove a single product.  
5. Use the **Delete All** button to remove all products (appears above the table if products exist).  
6. Use the **Search input** to filter by **Title** or **Category** as you type.
